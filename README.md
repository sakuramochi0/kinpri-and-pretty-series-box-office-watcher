## Overview

- [x] collector: https://mimorin2014.com さんから『KING OF PRISM -Dramatic PRISM.1-』のデータを集める。

- [x] visualizer: 販売数の変化や興行収入の合計などの知りたい情報をわかりやすく可視化する。
